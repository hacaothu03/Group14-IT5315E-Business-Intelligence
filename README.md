# Group14 Business Intelligence - House Price Prediction

This repository is organized by project module.

- `Business_DataGen/`: Module 1 data generation/context work.
- `EDA/`: Module 2 exploratory data analysis.
- `CleanData/`: Module 3 cleaned data outputs.
- `Module4_5_Modeling/`: Module 4 feature engineering and Module 5 model development.
- `Module6_7_Deployment/`: Module 6 deployment (FastAPI API + Streamlit valuation tool) and Module 7 monitoring (drift + performance dashboard, Evidently report).


For Module 6-7 deployment and monitoring, read:
